# img-slider-one
Generates an image slider panel with the given images.

### Syntax
```
create(['array of images' relative paths'], (parent selector))
```

Example

```
create([
    ./assets/image-one.jpg,
    ./assets/image-two.jpg,
    ./....
], '.slider-wrap')
```
